# Maquina-Expendedora
Esta maquina como dice el titulo, recreamos una maquina expendedora virtual utilizando codigos de programacion. 
Esta maquina posee multiples variaciones de galletas, papitas y dulces, lo cual permite al usuario ecoger la preferida debido a la cantidad de dinero que ingrese. 
De sobrar, el programa esta diseñado para devolver el sobrante al usuario o repetir la acción.






¿Que hace este programa?

-Simular una maquina expendedora.

-Hace que el usuario entre una cantidad de dinero.

-Deja que el usuario escoja su snack favorito entre las opciones que tiene.

-Se le entrega lo pedido.

-De sobrar dinero, se le preguntara al usuario si desea continuar o no.

-De escoger si, se le repetira el menu de opciones.

- De escoger no, finaliza la compra.




How to use?

-El ususario entra su dinero.

-Escoge el codigo del snack que quiera comerse.

-El snack sale por la parte de abajo de la maquina.

-Si entra dinero de mas, se le entrega.

-Si no, se termina la compra.
